# BOT AUTO TF
a cli tool to automate interactions in blockchain

## Features
-  **auto transfer**
- **auto swap**
- **show profile**

## Prerequisites
- Node.js (v14+)
- npm or yarn
- Etherium wallet testnet

## Instalation
Clone the repository:
```bash 
https://github.com/Gusma-crypto/gzbot-evm.git
cd gzbot-evm
```

Install the Dependencies:
```bash
npm install
```
